\chapter{Gestión de Riesgos}
\label{ch:riesgos}

La gestión de riesgos identifica, evalúa y planifica la respuesta a eventos inciertos que podrían
afectar la calidad del producto o el éxito del proyecto de pruebas. Se distinguen dos categorías:
\textbf{riesgos del producto} (relacionados con el software bajo prueba) y \textbf{riesgos del proyecto
de pruebas} (relacionados con la ejecución del sprint SQA).

La escala de evaluación utilizada es:
\begin{itemize}
    \item \textbf{Probabilidad:} Baja (< 30\%), Media (30--60\%), Alta (> 60\%).
    \item \textbf{Impacto:} Bajo (no bloquea pruebas), Medio (retrasa pruebas), Alto (bloquea pruebas o entregable).
    \item \textbf{Severidad = Probabilidad $\times$ Impacto:} Baja, Media, Alta.
\end{itemize}

\section{Resultados de la Mitigación al Cierre del Sprint}
\label{sec:resultados-mitigacion}

Al finalizar el sprint SQA (21 de julio de 2026), el estado de los riesgos identificados fue el siguiente:

\begin{itemize}
    \item \textbf{Error 500 en formatter IA:} \textit{Mitigado}. Se implementó chunking de 12\,000 caracteres
          en \texttt{FormatterAgent}. Verificado con textos de hasta 51\,000 caracteres (5 chunks).
    \item \textbf{Crash del chat sin contexto (BUG-003):} \textit{Resuelto}. Corregido por el equipo de
          desarrollo. Re-test exitoso en CP-14.
    \item \textbf{Tiempo de automatización:} \textit{Parcialmente mitigado}. Se completaron los 6 archivos
          de test pytest. La suite E2E Cypress quedó como deuda técnica documentada.
    \item \textbf{Disponibilidad de micrófono:} \textit{Mitigado}. CP-03 ejecutado sin micrófono (OK);
          CP-04 y CP-05 ejecutados con micrófono disponible (OK con bug menor).
    \item \textbf{Disponibilidad de API NVIDIA NIM:} \textit{Sin incidencias}. Las claves de API
          permanecieron válidas durante todo el sprint.
    \item \textbf{Pérdida de datos en editor:} \textit{Parcialmente mitigado}. BUG-004 corregido;
          BUG-005 (texto muy largo) documentado como \textit{abierto} para el siguiente sprint.
\end{itemize}
