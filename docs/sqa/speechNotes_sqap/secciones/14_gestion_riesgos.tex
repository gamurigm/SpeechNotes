\section{Resultados de Mitigación}
\label{sec:resultados-mitigacion}

\begin{itemize}
    \item \textbf{NVIDIA NIM:} Se utilizó la clave de API proporcionada. Se ejecutaron las pruebas de formateo y chat sin problemas.
    \item \textbf{Inestabilidad frontend:} Se documentó BUG-002 (referencia de React no encontrada).
    \item \textbf{Tiempo de automatización:} Se priorizaron las pruebas pt.~\#7 y pt.~\#8 como automatizaciones adicionales.
    \item \textbf{Micrófono:} CP-03 (sin dispositivo) se ejecutó sin incidencias.
    \item \textbf{Bugs conocidos:} Se reportaron 8 bugs en Jira.
\end{itemize}
