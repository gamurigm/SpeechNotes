ℹ️ Cómo funciona:
El cliente (realtime.py) importa AudioRecorderFactoryProvider y le solicita un recorder específico (ej: RecorderType.VAD). El provider consulta su registro de factories, obtiene la factory correcta (VADRecorderFactory), la llama para crear el producto (VADRecorder), y lo retorna al cliente.
Ejemplo de uso en realtime.py:

recorder = AudioRecorderFactoryProvider.create_recorder(
    RecorderType.VAD,
    config=audio_config,
    vad_config=vad_config
)
audio_chunks = recorder.record()
