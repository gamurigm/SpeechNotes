# Questions for the Maintainers

This document contains a list of questions for the maintainers of the SpeechNotes project. The purpose of these questions is to clarify any ambiguities and to gather more information about the project.

1.  **What is the long-term vision for SpeechNotes?** Are there any plans to add new features or to expand the scope of the project?
2.  **What is the current status of the project?** Is it in active development, or is it in maintenance mode?
3.  **Who are the target users of SpeechNotes?** Is it intended for personal use, or is it designed for a specific industry or use case?
4.  **What is the preferred method for contributing to the project?** Are there any coding style guidelines or contribution policies that I should be aware of?
5.  **Are there any plans to implement a CI/CD pipeline?** If so, which CI/CD tool would you prefer to use?
6.  **What is the current test coverage of the application?** Are there any plans to improve the testing suite?
7.  **What is the preferred secret management solution for the project?** Are there any security requirements that I should be aware of?
8.  **What are the main performance bottlenecks of the application?** Are there any plans to improve the performance of the application?
9.  **What are the main pain points for the users of the application?** Are there any plans to improve the user experience of the application?
10. **What is the purpose of the `temporal_docs` directory?** It is not clear from the documentation what this directory is used for.
